条件分岐

★一つの場合

  if(条件1){
    処理1
  }
  
  条件1を満たすときは、処理1を実行


★一つの式で処理を二つに分ける場合

  if(条件1){
    処理1
  } else{
    処理2
  }

・条件1を満たすときは処理1、
  満たさないときは処理2を実行


★二つの条件式で処理を分ける場合
  if(条件1){
    処理1
  } else if(条件2){
    処理2
  } else{
    処理3
  }
  
  if(条件1){処理1} else if(条件2){処理2} else{処理3}

・条件1を満たすときは処理1、
  条件2を満たすときは処理2、
  どちらも満たさないときは処理3を実行
