条件分岐

★一つの場合

  if(条件1){
    処理1
  }
  
  条件1を満たすときは、処理1を実行


★一つの式で処理を二つに分ける場合

  if(条件1){
    処理1
  } else{
    処理2
  }

・条件1を満たすときは処理1、
  満たさないときは処理2を実行


★二つの条件式で処理を分ける場合
  if(条件1){
    処理1
  } else if(条件2){
    処理2
  } else{
    処理3
  }
  
  if(条件1){処理1;} else if(条件2){処理2;} else{処理3;}

・条件1を満たすときは処理1、
  条件2を満たすときは処理2、
  どちらも満たさないときは処理3を実行

繰り返し処理

◆while文

引数の条件がtrueの間、波括弧内部の処理が繰り返される。
  while(条件){
   処理
  }

・do while文  do{最低1回は行われる処理}while(条件式);

引数の条件がtrue／falseに関わらず、最初の1回だけは必ず処理
その後、条件式を評価して、繰り返し処理を続けるか／抜けるか

do{
  最低1回は行われる処理
}while(条件式);


◆for文
決められた回数だけ繰り返す
！「繰り返す回数」を初期値で設定する
！「条件を満たす場合の処理」を記述する

引数の条件がtureなら
for (初期値; 条件式; 増減値){
  繰り返したい処理
}
